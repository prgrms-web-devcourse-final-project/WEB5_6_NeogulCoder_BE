## 💡 프로젝트 소개

Wibby는 스터디 모집부터 운영·종료까지 **스터디 운영 전반을 지원하는 서비스**입니다. <br>
단순한 스터디 모집을 넘어 **운영·협업·성과 관리**까지 Wibby에서 해결할 수 있습니다. <br>
실시간 소통, 일정 관리, 자기 PR 매칭을 통해 **자율적이면서도 효율적인 스터디 문화**를 만들어갑니다.

# 🌟 기능 소개
### [ 회원 ]
- 이메일 인증 및 소셜 로그인
- 회원가입 시 기본 PR 템플릿 자동 생성

### [ 마이페이지 ]
- 회원 정보 조회 및 수정, 탈퇴
- 내가 참여한 스터디 / 신청한 스터디 / 작성한 모집글 조회
- 개인 캘린더 관리
- PR 템플릿 관리

### [ PR 템플릿 ]
- 자기소개 / 지역 / URL 등 프로필 관리
- 버디에너지 및 리뷰 확인

### [ 관리자 ]
- 회원 관리
- 스터디 관리
- 모집글 관리

### [ 모집글 ]
- 모집글 작성 및 조회
- 댓글 작성 및 관리

### [ 스터디 신청 ]
- 스터디 신청 / 승인 / 거절
- 신청 내역 관리 및 알림

### [ 스터디 ]
- 스터디 생성 및 운영 관리
- 스터디장 위임 / 랜덤 위임 / 연장
- 스터디 출석 및 출석률 조회
- 커뮤니티 게시판 (게시글, 댓글)
- 게시글 AI 기반 퀴즈 생성
- 최적 시간 투표
- 팀 캘린더
- 팀 채팅

### [ 리뷰 ]
- 종료된 스터디 리뷰 작성

### [ 버디에너지 ]
- 리뷰 기반 점수 시스템
- 스터디 완료·팀장 보너스 점수
